# Burza ucebnic

Tento projekt ma za cil digitalizovat kazdorocni burzu ucebnic a umoznit tak snazsi prodej a vice uspesnych objednavek.

## Instalace

```sh
$ make
```

Pro instalaci je vyzadovano php 8.1 a vyssi, yarn a nodejs.
